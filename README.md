

# 프로젝트 소개 (미정)
자신의 생각을 글로 표현할 수 있는 공간을 제공한다. 글쓰기는 단순히 창작의 즐거움을 넘어서, 정신 건강에 매우 긍정적인 영향을 미친다고 알려져 있다. 마음속의 복잡한 감정이나 스트레스를 글로 풀어내는 과정은 자기 이해를 돕고, 감정을 정리하는 데 큰 도움이 된다.
